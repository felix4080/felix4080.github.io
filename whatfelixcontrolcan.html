FelixControl

ВОЗМОЖНОСТИ FELIXCONTROL

1. Контроль места на дисках, при нехватке места в чат Telegram отправляется уведомление

2. Контроль температуры процессора, при превышении допустимого значения в чат Telegram отправляется уведомление

3. Полностью настраиваемая. Можно настроить своего бота, чтобы он писал при ненормальных показателях в ВАШУ группу. Можно настроить, за какими дисками будет "слежка", при каком количестве свободной памяти отправляется уведомление. Можно настроить, при какой температуре процессора будет отправляться уведомление. И ещё: для тех, у кого "слежка" за несколькими компьютерами, есть возможность написать своё имя компьютера, которое будет в уведомлении

4. Логирование температуры и места на дисках
